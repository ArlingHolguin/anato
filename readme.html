Prueba Técnica Wordpress *Utilice el archivo anato.sql para crear su base de datos De acuerdo a la siguiente instalación de WordPress 1. Realice los ajustes necesarios para que el sitio vuelva a ser visible correctamente (solventar errores) 2. Cree un theme llamado anato 3. En el theme anato, cree una página de home con la siguiente estructura: 3.1 header: una fila (row) formada por 3 columnas con el siguiente ancho: columna 1=25%, columna 2=50%, columna 3=25%) Columna 1: colocar logo anato Columna 2: colocar un menú básico (home, nosotros, servicios, contacto) los enlaces de este menú pueden estar vacíos no redireccionan a ningún lado Columna 3: Link vacío llamado Buscar 3.2 Contenido: Ancho 100%, altura mínimo 500px, contenido (colocar el shortcode a generar en el punto #4) 3.3 Footer (pie de página): Ancho 100%, color de fondo: azul 4. Cree un Plugin que mediante un shortcode imprima un saludo "Hola ANATO" 5. Agregue un Custom post type llamado Agencias, cuyos campos sean: Nombre Agencia (text), Tipo de Agencia (select) con opciones: VyT, Mayorista, Oficina de representación, Operadora) Nota: para este punto tiene 2 opciones. Opción 1 (SIN PLUGIN) ideal ya que otorga más puntos en la prueba y Opción 2 (Con un plugin) esta opción otorgará menos puntos en la prueba.
